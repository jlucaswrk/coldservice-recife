# Instruções para Claude

## Deploy Automático

Sempre que o usuário pedir uma alteração no código, após concluir as modificações:

1. Fazer commit das alterações com mensagem descritiva em português
2. Fazer push para a branch `main`

O Vercel está configurado para deploy automático quando a branch `main` recebe atualizações.

## Projeto

- **Stack**: Next.js 14 + Tailwind CSS 4
- **Hospedagem**: Vercel
- **Idioma**: Português (BR)
